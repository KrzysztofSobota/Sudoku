Sudoku game

This game come from Japan and is a mathematical brain teaser.

Big square is made from 81 small square fields (9 rows, 9 columns) and it has got two main 9-elements diagonals. Furthermore red lines show additional division by 3x3 squares.

The task is filling all the space with digits 1-9 in a such way that in every:

  • row,
  • column,
  • main diagonals,
  • 3x3 square,
none of these digits will be repeated.

Example of properly filled board game is below - after click "START" button image will disappear and the sudoku game begun. In "Options menu" you can find type of the game (time on/off) and a few difficulty levels. Number of empty cells (squares) on the board depends on chosen level of the game.
